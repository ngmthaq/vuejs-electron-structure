# vue-electron-structure

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Deployment
- Open Git bash terminal
- Set up GH_TOKEN
```
export GH_TOKEN=<token_go_here>
```
- Publish to Github
```
npm run electron:publish
```
